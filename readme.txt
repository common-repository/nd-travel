== Description ==

= Welcome to Travel WP plugin =
This plugin is an useful system to manage all your travel packages.

== Installation ==

1. Install and activate the plugin.
2. Create the page “Search” and insert the schortcode [nd_travel_search_results] for see all packages with search filter
3. Go in the plugin settings ( Travel Plugin ), and fill all fields.
4. Create your packages and destinations.
